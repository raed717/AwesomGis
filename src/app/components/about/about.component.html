Project Overview

Mapping Critical Infrastructure for Digital Growth

This master's project focuses on mapping and analyzing critical infrastructure and fiber-optic needs in the Impruneta area of Italy. The main objective is to design an interactive Web GIS tool that allows users to visualize, analyze, and plan the deployment of the fiber-optic network while taking existing infrastructure and territorial constraints into account.

By using advanced geomatics technologies, the project aims to optimize decision-making for the digital development of the municipality. It relies on the collection, processing, and integration of geospatial data to identify priority areas, assess current coverage, and propose effective development scenarios.

This work contributes to the modernization of telecommunications infrastructure and supports the promotion of smart and sustainable territorial planning.
